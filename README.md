# 🩺 Tri Health Predictor

**Tri Health Predictor** is an AI-powered web application developed using **Streamlit** that predicts the likelihood of **Diabetes, Heart Disease, and Parkinson’s Disease** using trained machine learning models. The application offers a clean, interactive interface with visually appealing cards, icons, gradient styling, and personalized health guidance to enhance the user experience.

## ✨ Features

- Predicts the risk of **Diabetes**, **Heart Disease**, and **Parkinson’s Disease**
- Interactive user input through sliders, radio buttons, and dropdown menus
- Displays prediction results using color-coded cards with health recommendations
- Modern and responsive user interface with icons, gradients, and custom styling
- Machine learning models loaded efficiently using **Pickle**
- Ready for deployment on **Streamlit Cloud**

## 🛠️ Tech Stack

- **Frontend:** Streamlit, HTML, CSS
- **Backend:** Python
- **Machine Learning:** Scikit-learn, Pickle
- **Deployment:** Streamlit Cloud
